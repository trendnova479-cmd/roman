Tu Marca — Tema Shopify (minimal)

Archivos incluidos:
- layout/theme.liquid
- templates/index.liquid
- sections/* (header, hero, featured-products, features, newsletter, footer)
- assets/* (styles.css, theme.js, placeholders SVG)
- config/settings_schema.json

Instrucciones:
1) Descarga el ZIP desde: https://github.com/trendnova479-cmd/roman/archive/refs/heads/main.zip
2) En Shopify Admin > Online Store > Themes > Upload theme, sube el ZIP.
3) Abre "Customize" para ajustar textos, colecciones e imágenes.

Si quieres que adapte las secciones para usar colecciones concretas o añadir traducciones, dímelo y lo actualizo.
